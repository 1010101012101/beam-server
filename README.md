## Setup

1. Get Node and npm
2. Clone this repo
3. `npm install`
4. `npm install -g db-migrate`

## DB

DB: [Postgres](postgresql.org) through [node-postgres](https://github.com/brianc/node-postgres)

DB Migration: [node-db-migrate](https://github.com/db-migrate/node-db-migrate)
